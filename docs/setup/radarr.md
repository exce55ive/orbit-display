## Radarr

### URL

```
http://YOUR-SERVER-IP:7878
```

---

### API Key

Radarr → **Settings → General → API Key**

---

### Put It in Orbit

**Settings → Integrations → Radarr**

```json
{
  "radarr": {
    "url": "http://YOUR-SERVER-IP:7878",
    "api_key": "YOUR_RADARR_API_KEY"
  }
}
```
