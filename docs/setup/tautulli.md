## Tautulli

### URL

```
http://YOUR-SERVER-IP:8181
```

---

### API Key

Tautulli → **Settings → Web Interface → API Key**

If no key exists, click **Generate** to create one.

---

### Put It in Orbit

**Settings → Integrations → Tautulli**

```json
{
  "tautulli": {
    "url": "http://YOUR-SERVER-IP:8181",
    "api_key": "YOUR_TAUTULLI_API_KEY"
  }
}
```
