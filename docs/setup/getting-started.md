# Getting Started with Orbit

Orbit is a live desktop dashboard for Windows that shows real-time data from your home server services, all arranged in side-by-side panels on a single screen.

## What it does

Orbit pulls live data from services like Sonarr, Radarr, Plex, Jellyfin, Home Assistant, Spotify, and more — displaying everything at a glance without switching between browser tabs.

## System requirements

- Windows 10 or 11 (64-bit)
- Recommended: ultrawide monitor (34"+) or a dual-monitor setup
- Local network access to the services you want to monitor

## How to install

1. Download the Orbit installer from the provided link.
2. Run the installer and follow the setup wizard.
3. Launch Orbit from the Start menu or desktop shortcut.

## Panel types

| Panel | What it shows |
|---|---|
| Now Playing | Active Plex or Jellyfin streams |
| Sonarr | TV library, queue, recent downloads |
| Radarr | Movie library, queue, upcoming releases |
| Overseerr | Request counts and recent requests |
| NZBGet | Download queue and speed |
| Uptime Kuma | Monitor status for your services |
| Home Assistant | Light controls |
| Spotify | Now playing, controls, queue |
| Discord | Profile and quick voice channel access |
| SignalRGB | Lighting effects |
| System Monitor | CPU, GPU, RAM, uptime |
| Clock & Weather | Time, date, 5-day forecast |

## Add, remove, or reorder panels

- Open **Settings** (⚙ button in the bottom bar).
- Click **Add Panel** to choose a service and fill in its credentials.
- To remove a panel: click the trash icon next to it in the Settings panel list.
- To reorder: drag panels left or right in the Settings list.

## Switch pages

- **Keyboard:** Use the left/right arrow keys to move between pages.
- **Touch screens:** Swipe left or right.

## Open Settings

Click the **⚙** button in the bottom bar at any time to open the Settings window.

---

> **Tip:** If a panel shows "offline" after setup, open Settings, verify the URL and credentials, and click **Test Connection**.
