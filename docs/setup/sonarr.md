## Sonarr

### URL

```
http://YOUR-SERVER-IP:8989
```

---

### API Key

Sonarr → **Settings → General → API Key**

---

### Put It in Orbit

**Settings → Integrations → Sonarr**

```json
{
  "sonarr": {
    "url": "http://YOUR-SERVER-IP:8989",
    "api_key": "YOUR_SONARR_API_KEY"
  }
}
```
