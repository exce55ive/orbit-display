Drag & Drop Layouts

          
Rearrange panels by dragging. Save multiple layout profiles for gaming, media, or monitoring. Switch instantly.

        

        

          
Multi-Page Dashboard

          
Create multiple pages with different panel sets. Swipe or use arrow keys to navigate between them.

        

        

          
Toast Notifications

          
Get real-time desktop notifications when Sonarr grabs an episode, Radarr downloads a movie, or NZBGet completes.

        

        

          
Auto-Updates

          
One-click updates from within the app. Orbit checks for new releases automatically and installs seamlessly.
