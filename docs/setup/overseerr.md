## Overseerr

### URL

```
http://YOUR-SERVER-IP:5055
```

---

### API Key

Overseerr → **Settings → General → API Key**

---

### Put It in Orbit

**Settings → Integrations → Overseerr**

```json
{
  "overseerr": {
    "url": "http://YOUR-SERVER-IP:5055",
    "api_key": "YOUR_OVERSEERR_API_KEY"
  }
}
```
